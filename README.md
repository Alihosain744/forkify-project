# forkify project

forkify is used to enable users to search over recipes, see their ingredients, bookmark recipes, see the nutrition table of each recipe,
upload their own recipes, add recipe to a weekly planning meal, increase
or decrease number of servings for ingredients, users can use this to also
add the ingredients to a shopping list, for shopping the ingredients needs to buy.

this was an amazing project i have made through my learning javascript,
i have learned many things by building this project, like understanding
mvc architecture, modules and many more beneficial tools.
